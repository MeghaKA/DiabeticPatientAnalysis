# 🩺 Diabetes Prediction with Explainable AI (SHAP)

## 🎯 Overview  
This project focuses on predicting diabetes risk using machine learning models and enhancing interpretability using SHAP (Explainable AI).

The goal is not only to build accurate models, but also to understand why predictions are made, which is critical in healthcare applications.

---

## ⚙️ Tech Stack  
- Python  
- Pandas, NumPy  
- Scikit-learn  
- SHAP (Explainable AI)  
- Matplotlib  

---

## 🚀 What I Built  
- Developed classification models:
  - Decision Tree  
  - Support Vector Machine (SVM)  
  - Logistic Regression  
- Evaluated models using:
  - Accuracy  
  - Precision, Recall, F1-score  
  - Confusion Matrix  
- Applied SHAP for model interpretability  

---

## 📊 Explainability (SHAP Analysis)

SHAP was used to understand feature importance and model behavior.

📌 Key observations:

- General Health (GenHlth) has the strongest impact on predictions  
- High Blood Pressure (HighBP) significantly increases diabetes risk  
- BMI and Age show strong positive influence  
- High Cholesterol (HighChol) also contributes to higher predictions
